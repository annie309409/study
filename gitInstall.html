1. Git 파일 다운로드

https://git-scm.com/downloads

 

사이트에 접속해서 운영체제에 맞는 아이콘을 클릭합니다.

 

 

 

Standalone Installer과 Portable 두 가지가 있습니다.

Standalone은 실행파일을 다운 받고,

Portable은 무설치 버전 파일입니다. 즉, exe파일이 아닌 7z 압축파일이 설치됩니다.

 

저는 Standalone Installer 32-bit Git for Windows Setup 을 다운로드하였습니다.

3. Git Bash 사용법

 

 

Git을 사용하고자 하는 폴더에 들어가 오른쪽 마우스를 클릭합니다.

 

 

Git Bash Here을 클릭하면 해당 폴더에서 Git을 실행할 수 있습니다.
